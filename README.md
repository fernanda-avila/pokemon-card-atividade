# Projeto Pokémon Card - Atividade Framer Motion

## Como usar

1. Rode `npm install` para instalar as dependências.
2. Rode `npm start` para iniciar o servidor de desenvolvimento.
3. Abra `http://localhost:3000` no navegador para ver os cards.
4. O componente `PokemonCard` está em `src/components/PokemonCard.jsx`.
5. Desafio: adicione animações usando Framer Motion nos cards!

## Estrutura do projeto

- `public/assets/`: imagens dos pokémons.
- `src/components/PokemonCard.jsx`: componente do card.
- `src/App.jsx`: página principal que usa os cards.
- `src/index.js`: ponto de entrada.
